## 📌 Project Description
This project analyzes Git as an open-source software tool. It explores its origin, philosophy, Linux footprint, and practical usage through shell scripting. The project demonstrates real Linux interaction and automation using Bash scripts.

---

## ⚙️ Scripts Included

### 🔹 Script 1: System Information Script
Displays system details such as OS version, kernel version, current user, uptime, and date.

### 🔹 Script 2: FOSS Package Inspector
Checks whether Git is installed and displays package details using Linux commands.

### 🔹 Script 3: Disk and Permission Auditor
Analyzes important system directories and displays their size, permissions, and ownership.

### 🔹 Script 4: Log File Analyzer
Reads a log file and counts occurrences of a keyword like "error" or "warning".

### 🔹 Script 5: Open Source Manifesto Generator
Takes user input and generates a personalized open-source philosophy statement.

---

## ▶️ How to Run Scripts (Linux)

### Step 1: Give permission
```
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```

### Step 2: Run scripts
```
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
./script5.sh
```
